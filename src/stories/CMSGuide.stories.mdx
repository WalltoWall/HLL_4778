## Sections

Use the sidebar to the left to navgigate through different sections of the
documentation, or navigate through main sections found above.

### Documentation

General information about website content, its underlying services, and
management guides.

- **General**: Overview of managing the site and its content.
- **Prismic** CMS: Guides on using the site's content management system,
  Prismic.
- **Workflows**: Instructions on performing common content updates.

### Slices

Documentation regarding Slices, the reusable building blocks used to create
pages on the site.

### Document Types

While [Pages](/story/document-types-page--page) and
[Events](/story/document-types-event--page) will be the most common document
types used throughout the site, all other content is managed through a
collection of other document types. This includes content such as the site's
copyright text, navigation menus and video submissions.

Learn more about the different fields available to each document type in their
respective documentation within the **Document Types** section.

### Style Guide

Colors and typography used throughout the site are listed within the **Style
Guide** section. Use these specifications when creating new website content.
