# Navigation

The Navigation custom type represents groups of pages used to navigate the site.
This includes navigation lists displayed in the header.

The site only supports one navigation document: **Primary**. Any additional
Navigation documents will not have an affect on the site.

### Fields

Primary fields for this document. Do not edit. Editing will likely break the
navigation functionality on the site!
