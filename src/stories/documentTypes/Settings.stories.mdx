# Settings

The Settings document type represents general settings for the site. The
settings document has several tabs at the top that categorize various settings
to edit.

**Note**: Only one Settings document can be created. The master document
contains all content.

### Redirects

Visitors can be redirected from one URL to another by adding it to this list.

> **Note**: Only set **Is Permanent** if you are 100% sure the redirect will not
> need to be undone. You **cannot** undo a permanent redirect.

### Notification Bar

The Notification fields represents a notification displayed to all users at the
top of the page.

> **Note**: Displaying a notification will shift all content on a page down and
> reduce the user's visible area. **Please keep this message under 140
> characters**.
