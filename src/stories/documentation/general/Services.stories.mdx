# Services

All services used to run and maintain the site. The site uses several services
to host, store, and track website content and visitors.

#### General

- [**Netlify**](https://netlify.com): Static site hosting platform used to serve
  and build the site. (Free plan)

- [**Prismic**](https://prismic.io): Content management system used to edit
  content on the site. (Starter plan)

- [**JWPlayer**](https://www.jwplayer.com/): Video content management system
  used to host videos that are shown on the site.

#### Voting

- [**Firebase**](https://firebase.google.com/): Votes are stored using Google's
  Firebase (Firestore database) platform. (Free plan)

#### Analytics

- [**Google Tag Manager**](https://marketingplatform.google.com/about/tag-manager/):
  Website tag manager used to load Google Analytics and any other tracking
  services. (Free)

- [**Google Analytics**](https://marketingplatform.google.com/about/analytics/):
  Website analytics platform used to understand website visitors. (Free)
